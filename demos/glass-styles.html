Glass Effect Styles
本设计系统提供 5 种玻璃效果风格，从高透明液态玻璃到低透明毛玻璃，覆盖 Apple iOS 26、macOS、Microsoft Fluent Design 等多种设计语言。每种风格都有独立的 CSS 变量支持，可自由组合使用。

Liquid Glass

Apple iOS 26 液态玻璃效果。高饱和度模糊 + 顶部高光 + 折射光线，营造流动通透的液态质感。

blur: 24px saturate: 180% highlight refraction
.glass-card

Frosted Glass

macOS 风格毛玻璃效果。中等模糊度 + 低饱和度，呈现磨砂质感的柔和通透感，适合大面积背景区域。

blur: 20px saturate: 150% matte
.frosted-card

Acrylic

Microsoft Fluent Design 亚克力效果。高模糊 + 噪点纹理 + 色调叠加，模拟真实亚克力材质的颗粒感。

blur: 30px saturate: 200% noise tint
.acrylic-card

Crystal Glass

水晶玻璃效果。高透明度 + 强高光 + 棱镜折射线，呈现清澈透亮的宝石质感，适合高端展示场景。

blur: 16px saturate: 180% prism highlight
.crystal-card

Translucent

半透明玻璃效果。极低模糊 + 高透明度，最大程度保留背景内容可见性，适合信息密度高的界面。

blur: 8px minimal lightweight
.translucent-card

3D Depth Card
多层阴影 + 伪元素高光 + 噪点纹理，模拟真实 3D 深度的液态玻璃卡片。灵感来自 lucasromerodb/liquid-glass-effect-macos。

glass3d + glass-tilt

3D 深度卡片 + 鼠标倾斜交互。多层 box-shadow 营造真实深度，鼠标移动时卡片跟随倾斜，灵感来自知乎/墨迹的液态玻璃教程。

Rotating + Light Track

旋转高光 + 鼠标追光。conic-gradient 8s 循环旋转，鼠标移动时径向渐变跟随，灵感来自 rdev/liquid-glass-react 和 wxperia/liquid-glass-vue。
