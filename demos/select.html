XiaoP Liquid Glass Select

Auto-transformed dropdown components with glass morphism effects

Basic Select

All <select> elements with class "form-select" are automatically transformed into liquid glass dropdowns by liquid-glass.js. The original select is hidden and a custom glass component is rendered.

<select class="form-select">
  <option value="">Select...</option>
  <option value="cn">China</option>
</select>

Pre-selected Values

Dropdowns with pre-selected values show the active indicator (glowing dot) next to the selected option.

Excluding Auto-transform

Add class "no-liquid-glass" to any select element to keep the native browser dropdown.

<select class="form-select no-liquid-glass">
  <!-- This will NOT be transformed -->
</select>

Dynamic Select (Added at Runtime)

When adding select elements dynamically, call window.initLiquidGlassSelects() to transform them.

// After adding a select dynamically:
container.appendChild(selectElement);
if (window.initLiquidGlassSelects) window.initLiquidGlassSelects();

Size Variants

Liquid glass dropdowns support small, default, and large sizes via CSS classes.
