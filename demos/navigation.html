Glass Tab Bar

液态玻璃分段控件，活跃标签使用渐变背景 + 高光覆盖层，切换时带有弹性缓动动画。

Overview — This is the overview panel. The tab bar above uses liquid glass styling with backdrop-filter blur and saturate. The active tab features a gradient background with a highlight overlay created by a ::before pseudo-element, and transitions use cubic-bezier elastic easing for a spring-like feel.

Analytics — Analytics dashboard with real-time data visualization. All cards use the glass-card pattern: semi-transparent background + backdrop-filter blur + border highlight + inner shadow for depth. Hover effects include translateY lift and enhanced shadow.

Reports — Generated reports with export options. The glass effect is achieved through multiple layers: a blurred background layer, a tinted overlay, and a shine layer with inset box-shadow for the highlight line at the top edge.

Settings — Application settings and preferences. Each glass component follows the design token system with --lg-* CSS variables, ensuring consistent theming across light and dark modes via prefers-color-scheme.
