Glass Tab Bar

液态玻璃分段控件，活跃标签使用渐变背景，切换时带有弹性缓动动画。

Overview — This is the overview panel. The tab bar above uses liquid glass styling with the 4-layer structure: outer (edge refraction via SVG feDisplacementMap + mask-composite:exclude), cover (blur 2px + tint), sharp (4-side inset highlights), and reflect (soft inner reflections).

Analytics — Analytics dashboard with real-time data visualization. All cards use the liquid-glass class which auto-creates 4 child layers via JavaScript. The stat cards also have the glass-elastic class for elastic mouse-following stretch effect.

Reports — Generated reports with export options. The liquid glass effect achieves edge-only refraction through mask-composite: exclude on the outer layer, making the SVG displacement map visible only at the 5px border area.

Settings — Application settings and preferences. Each glass component follows the design token system with --lg-* CSS variables, ensuring consistent theming across light and dark modes via prefers-color-scheme.
