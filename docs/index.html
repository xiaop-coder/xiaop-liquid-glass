Quick Start

Include the CSS and JS files in your HTML. That's it — no build tools, no frameworks, no dependencies.

<!-- In your <head> -->
<link rel="stylesheet" href="dist/liquid-glass.css">

<!-- Before </body> -->
<script src="dist/liquid-glass.js"></script>

Minimal Example

<div class="glass-card" style="padding: 24px; max-width: 400px; margin: 100px auto;">
  <h2>Hello Liquid Glass!</h2>
  <p>This card has the liquid glass effect.</p>
  <button class="glass-button">Click Me</button>
</div>

Installation

npm

npm install xiaop-liquid-glass

CDN

<link rel="stylesheet" href="https://unpkg.com/xiaop-liquid-glass/dist/liquid-glass.css">
<script src="https://unpkg.com/xiaop-liquid-glass/dist/liquid-glass.js"></script>

CSS Variables

All variables use the --lg-* prefix and support automatic light/dark mode switching via prefers-color-scheme.

VariableDefault (Dark)Description
--lg-bg-gradient-1#0c0c1dBackground gradient color 1
--lg-card-bgrgba(255,255,255,0.1)Card background
--lg-card-borderrgba(255,255,255,0.18)Card border
--lg-glass-bgrgba(255,255,255,0.12)Glass surface background
--lg-text-primary#ffffffPrimary text color
--lg-primary#007affPrimary accent color
--lg-secondary#8b5cf6Secondary accent color
--lg-input-bgrgba(255,255,255,0.08)Input background
--lg-radius-md14pxMedium border radius

Dark Mode

Dark mode is automatic via prefers-color-scheme. All --lg-* variables are overridden in the light mode media query. You can also manually override any variable:

:root {
  --lg-primary: #007aff;
  --lg-card-bg: rgba(255, 255, 255, 0.1);
}

@media (prefers-color-scheme: light) {
  :root {
    --lg-primary: #0055cc;
    --lg-card-bg: rgba(255, 255, 255, 0.6);
  }
}

JavaScript API

All JS effects auto-initialize on DOMContentLoaded. For dynamically added content, call the re-initialization function:

// Auto-initialization (called automatically)
initAllSelects(); // Transform all <select class="form-select">
initMouseTracking(); // Mouse light tracking
initScrollBlur(); // Navigation scroll blur
initRippleEffect(); // Click ripple on buttons
initEntranceAnimations(); // IntersectionObserver entrance

// After adding dynamic content:
if (window.initLiquidGlassSelects) window.initLiquidGlassSelects();

// Exclude select from transformation:
// Add class "no-liquid-glass" to any <select>
