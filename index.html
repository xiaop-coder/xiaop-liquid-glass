XiaoP Liquid Glass

小P液态玻璃设计系统 — 纯 CSS + JS 实现 Apple iOS 26 液态玻璃效果。核心特征：SVG feDisplacementMap 边缘折射 + 四边内高光 + 活跃态高透切换。零依赖、跨框架、完全响应式，自带深色模式。

50+ 变量 35+ 组件 13 种 JS 交互 响应式

快速开始

在 HTML 中引入 CSS 和 JS 文件即可。无需构建工具、无需框架、零依赖。

<!-- In your <head> -->
<link rel="stylesheet" href="dist/liquid-glass.css">

<!-- Before </body> -->
<script src="dist/liquid-glass.js"></script>

最小示例

<div class="liquid-glass" style="padding: 24px; max-width: 400px; margin: 100px auto;">
  <h2>Hello Liquid Glass!</h2>
  <p>Edge refraction + 4-side highlights.</p>
  <button class="liquid-glass-btn">Click Me</button>
</div>

液态玻璃组件

液态玻璃（Liquid Glass）是 Apple iOS 26 的核心视觉语言。与 Glassmorphism（毛玻璃）的本质区别：液态玻璃采用4层结构——outer层使用 SVG feDisplacementMap 边缘折射（mask-composite: exclude 仅边缘可见）+ cover层极低模糊 + sharp层四边锐利高光 + reflect层柔和反射。JS 自动为 .liquid-glass 元素创建4层子元素。

技术定义

各类玻璃效果的技术定义与视觉特征

Effect定义核心视觉实现方式
Liquid GlassApple iOS 26 动态材质，4层复合渲染：边缘折射+低模糊覆盖+锐利高光+柔和反射边缘折射(mask-composite:exclude) + 四边高光 + 活跃态高透4-layer: outer(feDisplacementMap+mask) + cover(blur 2px) + sharp(inset shadow) + reflect(inset shadow)
Frosted GlassmacOS 磨砂玻璃，Glassmorphism 标准实现均匀模糊 + 低饱和度 + 磨砂质感backdrop-filter: blur() saturate()
Translucent极简半透明，最大背景可见性极低模糊 + 高透明度backdrop-filter: blur(8px) + 低透明度背景
Skeuomorphic拟物化玻璃，模拟真实玻璃物理特性真实反射 + 折射 + 阴影 + 高光多层伪元素 + 复杂 box-shadow + SVG 滤镜

深色模式

深色模式通过 prefers-color-scheme 自动切换。所有 --lg-* 变量在浅色模式媒体查询中被覆盖。你也可以手动覆盖任何变量：

:root {
  --lg-primary: #007aff;
  --lg-card-bg: rgba(255, 255, 255, 0.1);
}

@media (prefers-color-scheme: light) {
  :root {
    --lg-primary: #0055cc;
    --lg-card-bg: rgba(255, 255, 255, 0.6);
  }
}

JavaScript API

所有 JS 效果在 DOMContentLoaded 时自动初始化。动态添加内容后，调用重新初始化函数：

// Auto-initialization (called automatically)
initLiquidGlass(); // Auto-create 4-layer structure + SVG filter for .liquid-glass elements

// After adding dynamic content:
if (window.initLiquidGlassSelects) window.initLiquidGlassSelects();
if (window.initLiquidGlass) window.initLiquidGlass();

// 4-layer structure (auto-created by JS):
// .liquid-glass-outer → backdrop-filter: url(#liquid-glass-filter) + mask-composite: exclude (edge refraction)
// .liquid-glass-cover → backdrop-filter: blur(2px) + rgba(0,0,0,0.12) (glass cover)
// .liquid-glass-sharp → inset box-shadow (sharp edge highlights)
// .liquid-glass-reflect → inset box-shadow (soft inner reflections)
// .liquid-glass-content → z-index: 5 (actual content)

// Exclude select from transformation:
// Add class "no-liquid-glass" to any <select>
